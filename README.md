# Roguelike Dungeon Crawler Game

A simple, customizable roguelike dungeon crawler game built with HTML5 Canvas and plain JavaScript. Explore randomly generated maps, fight enemies, collect items, and level up to defeat the boss!

---

## Features

- Procedurally generated maps with walls, floors, enemies, items, and a boss.
- Player movement within map boundaries, revealing nearby tiles.
- Collect health items and better weapons to improve stats.
- Turn-based combat against enemies, including a boss.
- Gain experience points (XP) to level up.
- All game elements are randomly placed for each playthrough.
- Win condition: defeat the boss.

---

## User Stories Fulfilled

- Track health, level, and weapon.
- Pick up better weapons and health items.
- Random arrangement of items and enemies.
- Move freely within map boundaries.
- Discover items as you explore.
- Cannot move through enemies until defeated.
- Visibility is limited around the player.
- Combat involves turn-based damage with randomness.
- Defeating the boss results in victory.
- Challenging but winnable gameplay.

---

## How to Play

1. Use arrow keys (`↑`, `↓`, `←`, `→`) to move around the map.
2. Explore, fight enemies, and collect items.
3. Defeat the boss to win the game.
4. If health drops to zero, game over.

---

## Technologies Used

- HTML5 Canvas
- JavaScript (ES6+)
- CSS for styling
